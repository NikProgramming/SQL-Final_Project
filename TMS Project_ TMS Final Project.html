Program Introduction

This Program is used for the TMS (Transportation Management System). TMS is a critical part of the shipping and logistics industry. Allowing for the purchasing, scheduling, monitoring and billing of organizations who desire to have cargo shipped via truck or train.

The TMS project features to note are:

  • Account System: Manages user accounts.
  • Billing System: Checks to see if users have payed and generates a receipt.
  • Buyer System: Allows the buyer to request contracts and create orders.
  • Carrier System: Sets up what carrier is currently in use for the TMS specific orders.
  • Invoice System: Creates an invoice for the users.
  • Order System: Allows for customer to order items.
  • Planner System: Allows for the planner to select a carrier.

The UI Section

  • MainWindow: The main window of the program that display the delivery type, login, market place, and register for the TMS program.
  • Login: used to allows admins to log into the program.
  • Register: allows for new admins to be created.
  • DeliveryTimes: display the time the order is supposed to arrive.
  • MarketPlace: use to allow admins to get new contracts.
